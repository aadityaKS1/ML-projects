# TMDB Movie Analysis

## Overview
This project analyzes **The Movie Database (TMDB)** dataset to extract insights about movie popularity, revenue, and genres. The goal is to practice **Data Analysis, Data Cleaning, and Visualization** using Python.

---

## Dataset
- **File:** `movies.csv`  
- **Source:** TMDB dataset (5000+ movies)  
- **Features include:**  
  - `title` – Movie name  
  - `genres` – Movie genre(s)  
  - `popularity` – Popularity score  
  - `revenue` – Box office revenue  
  - `vote_average` – Average rating  
  - `vote_count` – Number of votes  
  - `budget` – Movie budget  
  - `release_date` – Release date  

---

## Libraries Used
- `pandas` – Data manipulation  
- `numpy` – Numerical operations  
- `matplotlib` – Plotting  
- `seaborn` – Advanced visualization  

---

## Project Steps
1. **Data Loading** – Read the CSV and inspect basic info.  
2. **Data Cleaning** – Handle missing values and parse JSON-like fields (genres).  
3. **Feature Selection** – Keep only relevant columns.  
4. **Exploratory Analysis & Visualization**  
   - Top 10 most popular movies  
   - Top 10 highest revenue movies  
   - Genre analysis
